# Test Management
A centralized test management system with version control.

## Features
* Store and manage test cases in a centralized repository
* Version and track test cases using Git
* Collaborate on test case creation and modification

## Usage
1. Create a new test case using the `TestCase` class.
2. Add the test case to the `TestManagement` system using the `add_test_case` method.
3. Get all test cases using the `get_test_cases` method.
4. Update a test case using the `update_test_case` method.
5. Delete a test case using the `delete_test_case` method.
6. Save the test management system to a JSON file using the `save_to_json` method.
7. Load the test management system from a JSON file using the `load_from_json` method.
