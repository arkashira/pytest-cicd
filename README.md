# 🚀 pytest-cicd

**Power CI/CD pipelines with seamless pytest integration.** pytest-cicd is a cutting-edge tool that streamlines your Continuous Integration and Continuous Deployment (CI/CD) workflows by leveraging the power of pytest.

## Why pytest-cicd?

* **🚀 Fast and reliable testing**: pytest-cicd ensures that your tests run quickly and accurately, reducing the risk of errors and improving overall pipeline efficiency.
* **📈 Data-driven insights**: Get actionable insights from your test results, enabling data-driven decision-making and continuous improvement.
* **🚫 Automated testing**: pytest-cicd automates testing, freeing up your team to focus on high-priority tasks and innovation.
* **📈 Scalable and flexible**: Easily integrate pytest-cicd with your existing CI/CD pipelines, making it a versatile solution for teams of all sizes.

## Getting Started

### Install pytest-cicd

```bash
pip install pytest-cicd
```

### Run pytest-cicd

```bash
pytest-cicd run
```

### Test pytest-cicd

```bash
pytest-cicd test
```

## Deploy

### Deploy to GitHub Actions

```bash
pytest-cicd deploy --github-actions
```

### Deploy to CircleCI

```bash
pytest-cicd deploy --circleci
```
