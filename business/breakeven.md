# breakeven.md

## Unit Economics & Break-even Analysis for pytest-cicd

### Cost per Active User
- **Compute Costs**: $0.10 per user per month
- **Storage Costs**: $0.05 per user per month
- **Bandwidth Costs**: $0.03 per user per month
- **Total Cost per Active User**: 
  - **Compute**: $0.10
  - **Storage**: $0.05
  - **Bandwidth**: $0.03
  - **Total**: **$0.18**

### Pricing Tiers
| Tier Name          | Price ($/mo) | Features                                                                 |
|--------------------|--------------|--------------------------------------------------------------------------|
| Basic              | $10          | Automated testing, CI/CD integration, Basic reporting                    |
| Professional       | $25          | All Basic features + Advanced reporting, Test analytics, 24/7 support   |
| Enterprise         | $50          | All Professional features + Custom integrations, Dedicated account manager |

### Customer Acquisition Cost (CAC) Range
- **CAC Estimate**: $50 - $100 per user
  - Marketing expenses, sales efforts, and onboarding costs contribute to this range.

### Lifetime Value (LTV) Estimate
- **Average Revenue per User (ARPU)**: 
  - Basic: $10
  - Professional: $25
  - Enterprise: $50
- **Average Customer Lifespan**: 24 months
- **LTV Calculation**:
  - Basic: $10 * 24 = **$240**
  - Professional: $25 * 24 = **$600**
  - Enterprise: $50 * 24 = **$1200**

### Break-even Users Count
- **Monthly Fixed Costs**: $1,000 (estimated for operations, support, etc.)
- **Break-even Formula**: Break-even Users = Monthly Fixed Costs / (Price per User - Cost per Active User)
  - Basic: $1,000 / ($10 - $0.18) = **100.18** → **101 users**
  - Professional: $1,000 / ($25 - $0.18) = **40.08** → **41 users**
  - Enterprise: $1,000 / ($50 - $0.18) = **20.00** → **20 users**

### Path to $10K MRR
- **Target MRR**: $10,000
- **User Distribution**:
  - **Basic Tier**: 
    - Price: $10
    - Users Needed: $10,000 / $10 = **1,000 users**
  - **Professional Tier**: 
    - Price: $25
    - Users Needed: $10,000 / $25 = **400 users**
  - **Enterprise Tier**: 
    - Price: $50
    - Users Needed: $10,000 / $50 = **200 users**

### Summary
To reach $10K MRR, we can target different tiers based on our marketing strategy, with the Professional tier being the most balanced in terms of user count and revenue generation.
